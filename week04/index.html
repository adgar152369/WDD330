Week 8: Forms

  • Back then, a form would be submitted, sent to server, and processed using a back end language
  • Now, it’s common to process the form info on the front end before it is sent to the server using JS

Form Properties and Methods

  • Form objects have properties and methods
  • submit() method will submit the form automatically
  • reset() method will reset the form controls back to initial values
  • action() property sets the action attribute, so it’s sent to the URL to the server

Form Events

  • Focus event occurs when an element is focused on

Form Validation

  • JS cannot be relied on for validation before sent to server because it can be modified
  • JS only to enhance the user experience
  • HTML5 API Validation by adding the required attribute
  • Add a custom form validation with JS
