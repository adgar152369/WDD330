Chapter 12: Canvas, SVG, and D & D

Canvas

  • Canvas can improve performance of websites
  • Manipulate pixels in images and videos
  • First, add canvas element to the page
  • Text between tags is only shown if browser doesn't support canvas
  • Height and Width must be set
  • Drawing happens via Canvas JS API

SVG

  • Scalable Vector Graphics
  • An XML-based markup language for describing two-dimensional based vector graphics
  • SVG images can be rendered any size without loss of quality

Which one to use?

  • The answer is, it depends on what you are doing.
  • Canvas can be quicker for games - no access to what's drawn on the canvas via the DOM
