Week 09 Notes

The Browser Object Model

  • Collection of properties and methods
  • Properties are made available through the window object

Going Global

  • Any global variable created is a property of the window object
  • In a browser environment, the global object is the window object

The Document Object

  • Each window object contains a document object
  • Contains methods and properties that deal with the page that is loaded into the window

Cookies

  • Small files that are saved locally on a user's computer
  • Keep track of user info, like history, choices, etc.
  • Form of a text file that contain name/value pairs

Timing Functions

  • Schedule the execution of a function at regular invervals or randomly
  • setTimeout() accepts a callback and number of milliseconds for its parameters
  • setInterval() will invoke the callback after every given number of milliseconds

The data- Attribute

  • Way of embedding data in a web page using custom attributes that are ignored by the browser
  • Private; not intended to be used by an external service
  • Can be used to ID particular elements
  • dataset property can be used to access any data- attributes it contains
  • Hyphens are replaced by camel casing while using the dataset object, accessing the properties of data-

HTML5 APIs

  • Give developers access to new features in hardware
  • Web storage API is useful without the restriction of cookies
  • The Geolocation API is used to obtain the geographical position of the device
  • Link to nearby places or measure speed at which the user is moving
  • Property of the navigator object

Web Workers

  • Allow processes to be run in the background
  • Use the Worker() constructor function to create a new worker
  • const worker = new Worker(js.js);
  • This function takes the name of another JS file as an argument
