Callback Functions

  • A callback function is a function passed as an argument to another function.
    This allows a function to call another function.
  • Useful for sorting arrays with numbers.

Hoisting

  • The interpreter moves variable and function declarations to the top of the scope, so it doesn't matter where they are declared.
  • Function are automatically hoisted if they are function declarations.
    This also means a declared function can be called before it's been defined.
